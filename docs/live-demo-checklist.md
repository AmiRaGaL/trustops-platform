# Live Demo Checklist

Use this smoke-test walkthrough for the deployed TrustOps demo:

1. Open the live demo: [https://trustops-platform-web.vercel.app/](https://trustops-platform-web.vercel.app/).
2. Log in as the moderator: `mod@trustops.dev` / `Password123!`.
3. Open the dashboard.
4. Open the report queue.
5. Open a report detail page.
6. Assign the report.
7. Add an internal note.
8. Take a moderation action.
9. Open audit logs and verify the action appears.

The public demo database should contain synthetic seed data only.
